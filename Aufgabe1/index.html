Mac: Terminal-Befehle für einfacheres Arbeiten

Was ist Terminal?

Ein Terminal ist textbasiert und dient als Befehlzeilenschnittstelle (Command Line Interface = CLI), in die Ihr Eure Befehle eingeben könnt. Eine Shell nimmt diese Befehle entgegen und weist das Betriebssystem an, sie auszuführen.

  1. Terminal app via "Launchpad" öffnen
  2. pwd - Print Working Directory
  3. ls - List
  4. ls -1 - List in eine Spalte
  5. clear
  6. ls -a - List mit versteckten Datein
  7. cd - Change directory
  8. mkdir (foldername)- Make directory
  9. cd .. - Change directory zurück
  10. touch - Eine Datei erstellen
  11. mv "file" "new place" - Datei verschieben
  12. cp "file" "new file" / cp -r file newFile - Datei kopieren
  13. cd~ - Zurück zum User-Ordner mit Tilde (Alt N)
  14. dig - abfrage des Domain Name Sytem
  15. exist - Das aktuelle Terminal beenden/verlassen.
  16. Ctrl + U: Löscht die Linie vom Cursorpunkt zurück zum Anfang.
  17. Ctrl + A: Bewegt den Cursor an den Anfrang der Zeile.
  18. Ctrl + E: Bewegt den Cursor an das Ende der Zeile.
  19. Pfeiltaste hoch/runter: Ermöglicht es Ihnen, die vorherigen Befehle zu durchsuchen.

Was ist Git?

Git ist ein verteiltes Versionskontrollsystem zur Verwaltung von Quellcode.

Ok, was ist also Versionskontrolle? Einfach ausgedrükt, ist die Versionskontrolle ein System zur Verfolgung von Änderungen an Dateien. Wenn wir Dateien ändern, zeichnet das Versionskontrollsystem jede Änderung auf und speichert sie. Auf diese Weise können wir jederzeit iene frühere Version unseres Codes wiederherstellen.
